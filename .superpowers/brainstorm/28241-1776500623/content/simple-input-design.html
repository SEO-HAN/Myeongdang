🎯 핵심 설계 원칙 (SV Senior PM)

1
세그먼트 자동 이동 — YYYY(4자리) 입력 완료 시 MM으로, MM(2자리) 완료 시 DD로 자동 포커스 이동. 키보드 한 번에 날짜 입력 완료.
2
실시간 절기 피드백 — 날짜 입력 즉시 "입춘 이전/이후" 안내 + 해당 사주 년도 표시. 사용자가 결과를 미리 이해.
3
시간 단독 입력 필드 — 24셀 그리드 대신 숫자 2자리 직접 입력 + 실시간 지시(地支時) 변환 배지. 인지 부담 ↓, 정확도 ↑.
4
결과 사주팔자 4기둥 명시 — 년/월/일/시주 한자·한글·오행 함께 표시. 일주(나)를 어두운 배경으로 강조.
5
입력 요약 카드 — Step 3에서 전체 입력값(년월일시) + 4기둥 요약 미리보기. 제출 전 확인 단계.
